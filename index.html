Contributing

We're always looking for new contributors. Here are some ways you can contribute:

- Opening Issues

Found a bug or a security risk in our project? Open a issue so we can get it fixed.

- Contributing Code

Want to help us by Coding? Open a pull-request, and we will give it a look.

- Documentation

Know everything about our project? Write some documentation to help other people know how it works.
